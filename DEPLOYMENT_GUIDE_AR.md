# دليل تركيب سكربت كلامنجي شات

لتركيب السكربت على الاستضافة الخاصة بك، اتبع الخطوات التالية:

### الخطوة 1: تجهيز الملفات
1. قم برفع كافة ملفات المشروع إلى المجلد الرئيسي في الاستضافة (مثل `public_html`).
2. تأكد من وجود ملف `.env` الذي يحتوي على روابط Supabase الخاصة بك.

### الخطوة 2: تثبيت الاعتمادات
إذا كانت الاستضافة تدعم Terminal، قم بتشغيل الأمر:
```bash
npm install
```

### الخطوة 3: بناء المشروع (Build)
قم بتحويل الكود إلى صيغة جاهزة للعمل:
```bash
npm run build
```
سيظهر لك مجلد باسم `dist` أو `build`؛ هذا هو المجلد الذي يجب أن يعمل عليه السيرفر.

### الخطوة 4: تفعيل مفتاح الأمان (License Key)
عند تشغيل السكربت لأول مرة، سيطلب منك "مفتاح الترخيص".
- اذهب إلى ملف `LICENSE_KEY.php` (أو الملف المخصص للمفتاح).
- تأكد أن المفتاح الموجود في الملف يتطابق مع المفتاح الذي ستدخله في واجهة التركيب.

### الخطوة 5: إعدادات قاعدة البيانات
- السكربت يعتمد على **Supabase**. تأكد من تفعيل جداول:
  - `user_profiles`
  - `messages`
  - `chat_rooms`
  - `site_settings`
  - `ads`

---
**ملاحظة**: السكربت مشفر بمفتاح أمان خاص، لا تشرك هذا المفتاح مع أحد.
